# 服务

自 `2.0` 版本起，Adyeshach 提供免费使用与付费版两种形式；付费版价格为 `300 CNY`。

1. 免费使用：可获取完整源码与功能文档，插件需<span style={{color: 'red'}}>自行构建</span>后部署。
2. 付费版（300 CNY）：购买后获得插件本体的<span style={{color: 'red'}}>下载与更新</span>，并享受<span style={{color: 'red'}}>优先</span>版本更新与问题解答。
3. <span style={{color: 'red'}}>因产品的特殊性，不提供任何形式的退款服务。</span>

<Callout type="info">

曾经购买过 Adyeshach 的用户，将保留为付费版。

</Callout>

## 体验

公开仓库中的源码与本文档站点内容，与付费版在功能说明上保持一致；未购买付费版时，请按[构建](start/build)文档自行编译插件。

联系作者购买付费版。
