## 介绍

Kether 是 TabooLib 框架中内置的脚本语言，由 [海螺先生](https://izzel.io) 创造。可以轻松实现诸多功能（如：发送动作栏或标题信息、改变玩家游戏模式、获取变量等等），它还拥有良好的拓展 API，能让其他开发者更加轻松地开发出自己的动作语句。
