# 玩家数据

指令入口为 `/chemdah data` 或 `/chpd`。

```
 Usage: /chemdahplayerdata
         ├── set <player> <key> <value>
         ├── add <player> <key> <value>
         ├── remove <player> <key>
         └── clear <player>
```

该命令用法十分直观，不做过多介绍。
