# 单位列表

列出当前世界中的所有公共单位，也包括属于你的私有单位。

```
/npc list
```

也可以使用来自 `1.x` 版本的 `/npc near` 别名。

页面长得和恢复命令差不多，会按照距离排序。

<img id="game_img" src="/img/command-list-1.png" width="600" style={{marginBottom: "1rem"}}/>

怕你忘记我再告诉你一次。

你可以点击牌子搜索任何属于单位的信息，比如类型、名称、数据值等。
